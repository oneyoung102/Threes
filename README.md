# Threes!

> 본 파일은 `<windows.h>` 라이브러리를 사용하여 windows 운영체제에서만 작동합니다.

> 고등학교 2학년 여름방학 때, **Threes**라는 게임을 모방하여 만든 게임입니다.

<hr/>

## 플레이

> **Threes.cpp**를 실행시켜 터미널에서 플레이합니다.

> **Max_num_score.txt** 파일은 플레이어의 최대 점수를 담고 있습니다.   
> **Num_name.txt** 파일은 최대 점수를 얻은 플레이어의 이름을 담고 있습니다.   
> **Num_set.txt** 파일은 설정 세팅 상태를 담고 있습니다.   
> **have_num.txt** 파일은 플레이어가 발견한 숫자 종류를 담고 있습니다.

1. 4x4 게임보드에는 1, 2, 3, 6 등의 숫자가 존재합니다. 
2. 1과 2를 합쳐 3을 만들 수 있습니다.
3. 3의 배수는 서로 같은 숫자끼리만 합칠 수 있습니다. (ex. 3+3 = 6, 12+12 = 24)
4. 가장 큰 수를 만드는 것이 목표입니다.

> 자세한 방법은 **Threes** 게임 규칙을 참고하세요.
